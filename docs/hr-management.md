# EPIC-018 Human Resource Management (HR)

Status

Planning

Priority

P1

Description

Module HR giúp nhà trường quản lý nhân sự, hồ sơ nhân viên, phân công công việc, chấm công và nghỉ phép. Module được thiết kế đơn giản, phù hợp với quy mô trường mầm non.

---

# HR-001 Employee Management

Quản lý nhân viên.

## Chức năng

- Danh sách nhân viên
- Thêm nhân viên
- Chỉnh sửa thông tin
- Nghỉ việc
- Khóa/Kích hoạt tài khoản
- Import Excel
- Export Excel

---

# HR-002 Employee Profile

Quản lý hồ sơ nhân viên.

- CCCD

- Trạng thái làm việc
